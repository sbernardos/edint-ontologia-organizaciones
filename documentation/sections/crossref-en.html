hierarchical leveldp back to ToC or Data Property ToC

IRI: http://vocab.linkeddata.es/datosabiertos/def/sector-publico/organizacion#hierarchicalLevel

This sequence identifies the relative hierarchical level of the organization within the entity to which it belongs. For the root of an administrative level, this field should be 0. For the main (highest level) unit of any Ministry, Autonomous Community, or Local Entity, the field will have the value 1. Beyond this, the number will be incremented as appropriate.
