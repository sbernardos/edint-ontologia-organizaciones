Introduction back to ToC

This ontology is part of the catalog of ontology being developed within the context of the Data Space for Smart Urban Infrastructures (EDINT). It was also identified as a relevant ontology for representing data managed by local entities in the FEMP 2019 Open Data guide: 40 datasets to be published by Local Entities, as described in the corresponding note on datos.gob.es, and is therefore included in the Vocabulary Catalogue of Ciudades Abiertas (Open Cities). This is a later version of that ontology.
